Use command "do run.do" to compile modules and simulate the design.


Files Description

Files in DDR3_Memory_Controller folder
1) ddr3_BFM.sv - ddr3 memory module.
2) ddr3_parameters.sv  -  ddr3 memory parameters module.
3) DDR3_Controller.sv - Design module.
4) DDR3_timing_parameters.sv - Memory Controller parameters and declaration of FSM states. 
3) Write.sv, Read.sv, counter_module.sv - Modules used by the design
5) communication_interface.sv - Creating Bus Functional Model
6) testbench.sv - Testbench containing randomization class, tasks of various test cases, program block and top level module.

Other files 
7) 1_Gb_DDR3_SDRAM.pdf - micron data sheet  
8) Presentation.pdf - Contains presentation given in the class
9) Report.pdf - Report of the project.
